### 1.1.1. Installation avec **_Nodejs_** et **_npm_**

`npm install`

### 1.1.2. Informations complémentaires

## 1.2. Traitement du `CSS`

### 1.2.1. Compilation

Compilation du `CSS` avec :

`npm run sass`

### 1.2.2. Nettoyage du `CSS`

`npm run purge`

### 1.2.3. Ajout des préfixes vendeurs
